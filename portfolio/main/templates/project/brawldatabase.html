BrawlDatabase is a website where visitors can view true combos from the platform fighting game Brawlhalla. The concept is relatively similar to ComboCodex, however the website does not feature users due to the simplicity of its content. BrawlDatabase was the first project I started working on and is what got me into web development. Early work on it started in December of 2022, but it was never completed. The project was rebuilt from the ground up in May of 2023 and hosted then. Most recently, the website was completely redone in October 2024, and is updated whenever Brawlhalla is changed. The website is still relatively active with around 20,000 requests daily.
