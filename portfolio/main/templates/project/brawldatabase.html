BrawlDatabase is a website where visitors can view true combos from the platform fighting game Brawlhalla. The concept is similar to ComboCodex, but BrawlDatabase is simpler and doesn't have user accounts. This was the first project I started working on, igniting my interest in web development. I began initial work on it in December 2022, though it wasn't completed at the time. The project was rebuilt from scratch and hosted in May 2023, with the most recent overhaul occurring in October 2024. The site remains active, averaging around 20,000 requests daily, and is updated whenever Brawlhalla receives changes.
