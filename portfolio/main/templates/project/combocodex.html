ComboCodex is a website where users can share and view "team combos" from the platform fighting game Brawlhalla. The target audience is relatively niche, however the website still averages around 25,000 requests daily, has nearly 1000 registered users, and has over 10,000 total uploaded videos taking up around 50GB of space. While it is not the first project I started working on, ComboCodex was the first project I completed and hosted. It was initially made in February 2023, completely remade in July of 2023, and received a UI overhaul in December 2023. There are currently plans to completely remake the website to reflect my progress as a developer.

The website also features a small API used to connect with a discord bot, as the concept of ComboCodex originally started as a discord server and the server itself still has many active members that use the bot to interact with the website.
