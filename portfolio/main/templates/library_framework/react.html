React is a staple frontend framework. React uses JSX, a syntax extension for Javascript, to allow developers to easily create and use components throughout their app. It makes use of a virtual DOM to compare and seamlessly update the actual DOM, making for a clean interface. Through a variety of hooks, React makes it easy to manage things like state and re-renders of elements. While I likely won't use React in my own personal projects (for now, at least), I think it's incredibly important to have a grasp on the foundations of it due to its immense popularity and usability in large projects. For now my experience lies simply in small projects (like the infamous todo app) and playing around with it.
