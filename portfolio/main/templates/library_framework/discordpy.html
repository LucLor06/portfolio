Discord.py is a Python library designed to simplify interactions with Discord’s API, making bot creation much easier. Before diving into web development, I was focused on making Discord bots, as I spent a lot of time using Discord. I even ran a server with over 10,000 members that functioned like a wiki. Over time, the constraints of Discord's platform led me to build my own website (BrawlDatabase) to share information. Nowadays, I typically create bots that are integrated with my project APIs rather than standalone Discord bots.
